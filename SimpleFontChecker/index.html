Help

  • Select View: Switch between Character List, Preview, and Help.
  • Preview View: Choose between font block and scrollers in the Preview tab.
  • PNG File: Load a font sprite sheet in PNG format.
  • Zoom: Scale the Character List and ticker tiles.
  • Start Scroller: Toggle the ticker animation when Preview is set to scrollers.
  • Frame: Outer padding of the sprite sheet before tiles start.
  • Gap: Spacing between tiles inside the sprite sheet.
  • Canvas Width/Height: Size of the random preview canvas.
  • Grid Width/Height: Size of a single tile in pixels.
  • Tile Limit: Limit how many tiles are shown/used (0 = no limit).
  • Ignore Chars: Comma-separated tile indices to skip.
  • Save/Load Settings: Store or restore your current setup.
  • Selection: In Character List view, click the image to set the grid box and use arrows to nudge it.
